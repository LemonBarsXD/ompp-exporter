# osu!mania PP exporter

Python script that automatically scans your `osu!` folder, calculates the Performance Points (PP) and Star Ratings for all your mania maps, and exports them to a cleanly sorted CSV file.

## Prerequisites
To run this on a new computer, you need two things:
1. **Python 3** installed (make sure to **"Add Python to PATH"**).
2. The **maniera** package.

## Installation & Setup
1. Download `main.py` to your computer.
2. Open a Command Prompt (cmd) or PowerShell.
3. Install the required library by running:

   ```bash
   pip install maniera

## How to Run
Simply open a terminal in the folder where you saved the script and run:
   ```bash
   python main.py
